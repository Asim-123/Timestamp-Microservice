Timestamp Microservice

Convert dates to Unix timestamps and UTC strings

Try it out

Examples:

  • /api/2015-12-25
  • /api/1451001600000
  • /api/ (current time)

Result

Error

API Usage

Endpoint

GET /api/:date?

Parameters

  • date (optional): A valid date string or Unix timestamp

Response Format

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Error Response

{
  "error": "Invalid Date"
}
